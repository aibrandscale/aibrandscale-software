# Project: aibrandscale-software

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma + PostgreSQL
- NextAuth
- Stripe
- OpenAI API

## Behavior rules
- Always write complete files. Never truncate code or use `// ...` placeholders.
- When creating components, always include the full implementation (imports, types, JSX, exports).
- Prefer small, focused commits — one logical change per commit.
- When stuck on an error, try at most 3 distinct approaches before stopping to ask.
